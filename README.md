# Smart-Greenhouse-Monitoring-and-Control-System
Full-stack IoT greenhouse system that monitors and controls environmental conditions (temperature, moisture, humidity) in real time.

## Hardware
* This was designed with the Arduino UNO R4 board, but it should work with other Arduino devices as well

* Breadboard

* Capacitive Soil Moisture Sensor

* DHT22 Temperature/Humidity Sensor

## Set up
Simply connect the Arduino ports to the DHT22 sensor and the soil moisture sensor using the breadboard.
